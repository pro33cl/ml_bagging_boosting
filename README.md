# ml_bagging_boosting
Notebook que desarrolla un flujo completo de Bagging y Boosting para un problema predictivo en la industria de viajes. Incluye limpieza y transformación de datos, análisis exploratorio, selección de características, modelos candidatos, RandomForest, Lasso, XGBRegressor, tuning con GridSearchCV y evaluación con métricas de regresión.
